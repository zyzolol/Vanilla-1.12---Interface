History

 

1.10.26 – 04 / 03 / 2006 – JIM the Inventor

 

o       The previous version misplaced the Choose Effects window.  This has been fixed.

o       Pulse is the third JIM AddOn to feature “Load On Demand Configuration”.

* Players may notice that the AddOn loads faster and takes up less memory.

* AddOn authors should check out JIM_toolbox\LoadOnDemandConfig.lua if they are curious about this technique.

o       JIM gets artistic!  Pulse’s icon has changed once again.  The new style will be reflected in all of JIM’s future AddOns.

o       The “legacy code” that updates version 1.9.16 to 1.9.17-1.10.25 has been removed.  If you have a version that pre-dates 1.9.17, please run version 1.9.17 before downloading any more recent versions.  This change only affects Choose Effects, so if you don’t understand any of this, it’s no big deal. :-)

 

1.10.25 – 03 / 28 / 2006 – JIM the Inventor

 

o       Updated the TOC number to 11000 (latest version of WoW).

o       Fixed a bug that thought all your bags were the same size as the second one.  (This could cause problems with item use.)
